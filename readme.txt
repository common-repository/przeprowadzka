== Installation ==

Po zainstalowaniu pluginu ustaw odpowiednie przekierowanie na starym serwerze. Zobacz
http://blog.ljasinski.pl/komputery/przeprowadzka-nie-musi-bolec/, aby znaleźć gotowe
skrypty PHP i .htaccess.

Na razie żadne opcje konfiguracyjne nie są dostępne. Musisz ręcznie porzeźbić
plik ljpl-przeprowadzka.css w katalogu pluginu

= Dlaczego okienko mi się nie wyświetla? =

Sprawdź czy ustawiłeś dobrze przekierowanie
